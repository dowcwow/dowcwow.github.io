Version 1.1.0

New features and QOL improvements

  • New platform type: disappearing platforms! Look for these when you are further in your run!
  • Upgrade spawns are now guaranteed after certain scores.
  • You now spawn with 3 hearts instead of 2.
  • Fixed the issue where the ground looked separated.
  • Lifesaver perk will now only appear 3 times in a run.
  • OP perk now only saves the previously acquired perks, and lasts for the next 3 runs (even if you close the game).
  • Removed the "permanent perk" text on certain upgrade cards.
